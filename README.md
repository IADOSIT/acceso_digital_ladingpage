## 📦 Despliegue rápido

### Opción 1 — GitHub Pages (gratis, en minutos)
1. Sube el contenido a un repositorio GitHub público
2. Ve a **Settings → Pages → Source → main branch / root**
3. ¡Listo! Tu sitio estará en `https://tu-usuario.github.io/acceso_digital_landingpage`

### Opción 2 — Netlify (gratis, dominio personalizado)
1. Arrastra la carpeta a [netlify.com/drop](https://netlify.com/drop)
2. O conecta tu repositorio GitHub para deploys automáticos

### Opción 3 — Tu propio servidor
Sube todos los archivos a la carpeta pública de tu hosting (public_html, www, htdocs).  
No requiere backend ni base de datos — es HTML/CSS/JS puro.
